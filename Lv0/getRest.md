# 나머지 구하기

## 📌 문제 설명

정수 num1, num2가 매개변수로 주어질 때, num1를 num2로 나눈 나머지를 return 하도록 solution 함수를 완성해주세요.

### 제한 조건

- 0 < num1 <= 100
- 0 < num2 <= 100

### 입출력 예

| num1 | num2 | result |
| ---- | ---- | ------ |
| 3    | 2    | 1      |
| 10   | 5    | 0      |

# 🧐 접근

연산자를 사용해서 바로 연산.

```java
class Solution {
    public int solution(int num1, int num2) {
        return num1 % num2;
    }
}
```

# 💡 풀이

나머지 연산자를 사용해서 나머지를 바로 return.
