# 두 수의 합

## 📌 문제 설명

정수 num1과 num2가 주어질 때, num1과 num2의 합을 return하도록 soltuion 함수를 완성해주세요.

### 제한 조건

- -50_000 < num1 <= 50000
- -50_000 < num2 <= 50000

### 입출력 예

| num1 | num2 | result |
| ---- | ---- | ------ |
| 2    | 3    | 5      |
| 100  | 2    | 102    |

# 🧐 접근

`+` 연산자를 사용

```java
class Solution {
    public int solution(int num1, int num2) {
        return num1 + num2;
    }
}
```

# 💡 풀이

`+` 연산자로 연산한 값을 return.
